Open Sea Lab

Open Sea Lab is a web application which enables scientists and researchers to explore marine data via a clear map interface.

Our #oSoc18 team is working on this visualization. Examples of data displayed on the map are types of seafloor, chemical levels in the water, fisheries,... Right now this data is available on different sites but being able to see all of them on one map will make it easier to compare and analyze the data. This web application is aimed at policymakers, scientists, researchers and businesses related to marine life.
